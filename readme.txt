Внимание!
* Файлы курса обновлены. В настоящее время не требуется установка Live Reload, всё работает через Gulp автоматически. Достаточно установить пакеты и запустить "gulp".
* При определении размера шрифтов в настоящее время не следует использовать функцию em(). Достаточно просто указать размер в px. Смотрим готовый результат в ready_html.
* В настоящее время достаточно использовать только woff2 шрифты (см. _fonts.sass)
